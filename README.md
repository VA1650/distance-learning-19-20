Дистанционное обучение — Web Archive (20XX)

    ⚠️ Project Status: Legacy / Archive
    Этот проект выложен в демонстрационных целях как пример раннего опыта в веб-разработке. На данный момент поддержка проекта прекращена, а база данных не включена в репозиторий.

📝 Описание

Это сайт для дистанционного обучения, который я разрабатывала в 2019-2020 годах. Основной фокус был на изучении PHP, JS и основ проектирования интерфейсов. 
🛠 Стек технологий 

    Frontend: HTML5, CSS3, JS 

    Backend: PHP

    Database: MySQL (на данный момент структура таблиц была утрачена)

💡 Что внутри (Features)

    Полноценная система авторизации пользователей. 

    Хранение паролей в хэшированном виде

    Личный кабинет с возможностью добавления своих лекций и заданий с помощью ckeditor для учителей.

    Лекции, тесты и иные практические задания для учеников

    Конвертер физических величин 
    
📂 Структура проекта

    /admin — Логика управления админкой.

    /css — Стили CSS

    /includes — Логика обработки запросов к БД и работы AJAX.

    /tpl — Элементы для блокового подключения и отображения на сайте

    index.html — Главная страница.

    index.php — Точка входа в приложение
